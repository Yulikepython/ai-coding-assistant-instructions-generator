🤖 AI Coding Assistant Instructions Generator

プロジェクトの特性に応じた AI開発ツール（GitHub Copilot、Cursor、Claude Code等）の指示文テンプレートを生成します。必要な項目を選択して、カスタマイズされた指示文を作成してください。

⚠️ 重要な注意事項

  • 生成されたテンプレートはあくまでベースです。プロジェクト固有の要件に応じてカスタマイズしてください。
  • 指示文は継続的に改良されています。定期的に最新版を確認することをお勧めします。
  • 複雑なプロジェクトでは、段階的に指示を追加していく方が効果的です。
